# IFSA Website Project Guide

## Project Overview
* **Name:** Indian Fit Sports Association (IFSA)
* **Goal:** A professional platform for fitness training and karate instruction.
* **Stack:** [Add your stack here, e.g., React, Tailwind CSS, or Plain HTML/JS]

## Build & Development Commands
* **Install dependencies:** `npm install`
* **Run dev server:** `npm run dev` or `live-server`
* **Build for production:** `npm run build`

## Coding Standards
* **Style:** Use clean, semantic HTML5 and responsive CSS (mobile-first).
* **Naming:** Use camelCase for JavaScript variables and kebab-case for CSS classes.
* **Images:** Always include `alt` tags for accessibility, especially for karate action shots.

## Project Context
* The site needs to handle memberships, course details for karate, and contact forms for new students.
* Focus on a high-energy, "fitness-forward" aesthetic.
